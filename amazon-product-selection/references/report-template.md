## 格式规范

### 表格
- 使用 Markdown 标准表格格式
- 数字右对齐，文本左对齐
- 关键指标加粗显示（如 **需供比**、**增长率**）

### 数字格式
- 整数：使用千位分隔符（如 9,861）
- 小数：保留 1-2 位（如 69.0、0.544）
- 百分比：使用 % 符号（如 +14.7%、-8.4%）

### 列表
- 使用无序列表（-）表示行动建议
- 使用复选框（- [ ]）表示待办事项
- 每条建议简洁明了，不超过一行

### Obsidian 兼容性
- 使用标准 Markdown 语法
- 表格使用 `|` 分隔
- 代码块使用 ``` 包裹
- 链接使用 `[text](url)` 格式
