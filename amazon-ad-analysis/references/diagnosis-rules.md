# 问题诊断规则库

## 一、销售情况评估规则（新增）

### 规则S1：销售情况判断

**评估维度**：

| 维度 | 差 | 好 |
|------|-----|-----|
| 销量趋势 | 环比下降>10% | 环比增长或持平 |
| 订单量 | 低于店铺平均 | 高于店铺平均 |
| 广告占比 | 过高(>60%)或过低(<20%) | 适中(20-60%) |

**判断标准**：
- 3个维度中2个为"差" → 销售情况判断为**差**
- 3个维度中2个为"好" → 销售情况判断为**好**

**特殊场景**：
- 新品期（上架<30天）：销售情况自动判断为**差**（需要提升）
- 爆款期（销量排名前10%）：销售情况自动判断为**好**（需要维护）

---

## 二、优化难度评估规则（新增）

### 规则D1：优化难度判断

**评估维度**：

| 维度 | 低（容易优化） | 高（难以优化） |
|------|----------------|----------------|
| ACOS | <40%，有明确优化空间 | >50%，结构性问题 |
| CVR | 低于店铺平均，有改进空间 | 正常或高于平均 |
| CPC | 高于店铺平均，可调整 | 已是最低水平 |
| 搜索词 | 有高ACOS词可否定 | 词本身竞争激烈 |
| 广告结构 | 单一，可调整 | 已较完善 |

**判断标准**：
- 5个维度中3个为"低" → 优化难度判断为**低**
- 5个维度中3个为"高" → 优化难度判断为**高**

**特殊场景**：
- ACOS>80%：优化难度自动判断为**高**（需要重构）
- CVR<5%：优化难度自动判断为**低**（Listing有明显改进空间）

---

## 三、策略方向选择规则（新增）

### 规则R1：策略矩阵

| 销售情况 | 优化难度 | 策略方向 | 优先级 |
|----------|----------|----------|--------|
| 差 | 低 | **进攻型** | 优先提升销售 |
| 差 | 高 | **防守型** | 控制亏损，等待时机 |
| 好 | 低 | **优化型** | 优化利润，提升效率 |
| 好 | 高 | **巩固型** | 保护优势，稳健发展 |

### 规则R2：方案类型匹配

**进攻型方案**（销售差 + 优化容易）：
- 快速止血：暂停高ACOS广告
- 精准提效：调整出价、否定词
- Listing优化：提升CVR
- 预算重分配：集中资源到高效词

**防守型方案**（销售差 + 优化困难）：
- 控制亏损：降低无效花费
- 维护现有：保持核心词投放
- 等待时机：监控市场变化
- 长期布局：准备Listing优化

**优化型方案**（销售好 + 优化容易）：
- 效率提升：降低ACOS
- 利润释放：减少不必要花费
- 精准投放：优化关键词结构
- 品牌建设：测试SB广告

**巩固型方案**（销售好 + 优化困难）：
- 保护优势：维持现有广告架构
- 稳健放量：小幅增加预算
- 竞品监控：防止竞品冲击
- 长期布局：品牌护城河建设

## 四、ACOS问题诊断

### 规则1：ACOS过高（>35%）

**诊断路径**：
```
ACOS过高
├── CPC过高（>品类均值1.5倍）
│   ├── 关键词竞价过高 → 降低竞价10-20%
│   ├── 质量得分低 → 优化相关性
│   └── 竞争激烈 → 转向长尾关键词
├── CTR过低（<0.2%）
│   ├── 主图不吸引 → 更换主图
│   ├── 标题不相关 → 优化标题
│   └── 价格无竞争力 → 调整价格
├── CVR过低（<5%）
│   ├── Listing质量差 → 优化描述、图片
│   ├── 评论少或评分低 → 积累评论
│   └── 配送不吸引 → 使用FBA
└── 客单价过低
    ├── 产品定位问题 → 考虑提价
    └── 组合销售机会 → 设置捆绑促销
```

### 规则2：ACOS波动大

**诊断标准**：7日内ACOS标准差>均值的30%

**可能原因**：
1. 关键词竞争波动 → 使用固定竞价而非动态竞价
2. 预算不稳定 → 设置每日预算上限
3. 季节性波动 → 对比历史同期数据

---

## 五、流量问题诊断

### 规则3：曝光量不足

**诊断标准**：日均曝光<1000（SP广告）

**可能原因与解决方案**：
| 原因 | 诊断方法 | 解决方案 |
|------|----------|----------|
| 预算不足 | 花费=预算 | 增加日预算20-50% |
| 关键词覆盖少 | 关键词数量<20 | 拓展关键词至50+ |
| 竞价过低 | IS<50% | 提高竞价10-20% |
| 关键词不相关 | CTR<0.1% | 优化关键词匹配方式 |

### 规则4：CTR过低

**诊断标准**：CTR<0.2%（SP广告）

**可能原因与解决方案**：
| 原因 | 诊断方法 | 解决方案 |
|------|----------|----------|
| 主图不吸引 | 对比竞品主图 | 更换主图，测试3-5张 |
| 标题不相关 | 关键词与搜索意图不匹配 | 优化标题前80字符 |
| 价格无竞争力 | 价格>竞品均值10% | 调整价格或突出价值 |
| 评分低 | 评分<4.0 | 积累评论，提升评分 |
| 配送不吸引 | 非FBA或配送时间长 | 使用FBA |

### 规则5：流量来源失衡

**诊断标准**：移动端流量<50%（大多数品类）

**可能原因**：
1. 主图在移动端显示效果差 → 优化主图尺寸和清晰度
2. 标题在移动端被截断 → 前80字符包含核心信息
3. A+页面在移动端加载慢 → 优化图片大小

---

## 六、转化问题诊断

### 规则6：CVR过低

**诊断标准**：CVR<5%（因品类而异）

**诊断路径**：
```
CVR过低
├── Listing质量问题
│   ├── 标题不清晰 → 优化标题结构
│   ├── 图片数量不足 → 补充至7张以上
│   ├── 描述不详细 → 优化五点描述
│   └── A+页面缺失 → 创建A+页面
├── 信任度问题
│   ├── 评论数量少（<50条）→ 积累评论
│   ├── 评分低（<4.0）→ 改善产品质量
│   └── 品牌知名度低 → 品牌推广
├── 价格问题
│   ├── 价格过高 → 调整价格或突出价值
│   ├── 促销不吸引 → 设置优惠券或促销
│   └── 隐藏费用 → 突出免运费
└── 配送问题
    ├── 配送时间长 → 使用FBA
    ├── 库存不足 → 补充库存
    └── 退货率高 → 改善产品质量
```

### 规则7：加购率高但下单率低

**诊断标准**：加购率>10%但下单率<3%

**可能原因**：
1. 价格在最后一步劝退 → 设置优惠券降低感知价格
2. 配送费用高 → 突出免运费或FBA
3. 库存紧张提示 → 及时补货
4. 支付流程问题 → 检查Listing设置

### 规则8：回头客少

**诊断标准**：重复购买率<10%（非消耗品正常）

**可能原因**：
1. 产品质量问题 → 改善质量
2. 售后服务差 → 优化售后流程
3. 缺乏忠诚度计划 → 设置订阅优惠
4. 竞品替代多 → 差异化定位

---

## 七、广告结构问题

### 规则9：广告类型失衡

**诊断标准**：
- SP占比>90% → 过度依赖SP
- SB占比<10% → 品牌建设不足
- SBV占比<5% → 视频广告缺失

**优化建议**：
1. 增加SB广告预算至20-30%
2. 创建SBV视频广告
3. 测试SD再营销广告

### 规则10：广告组效率差异大

**诊断标准**：同一ASIN的不同广告组ACOS差异>20个百分点

**优化建议**：
1. 分析高效广告组的关键词和出价策略
2. 将低效广告组的预算转移至高效组
3. 暂停持续低效的广告组

---

## 八、时间维度问题

### 规则11：销量持续下滑

**诊断标准**：连续7天销量环比下降>10%

**诊断路径**：
```
销量下滑
├── 流量下降
│   ├── Sessions下降 → 检查广告预算和竞价
│   ├── 曝光下降 → 检查关键词排名
│   └── CTR下降 → 检查主图和标题
├── 转化下降
│   ├── CVR下降 → 检查Listing和评论
│   ├── 评分下降 → 检查产品质量
│   └── 竞品降价 → 调整价格策略
└── 外部因素
    ├── 季节性波动 → 对比历史数据
    ├── 节假日影响 → 调整营销策略
    └── 竞品新品上市 → 差异化竞争
```

### 规则12：周末效应异常

**诊断标准**：周末销量与工作日差异>50%

**可能原因**：
1. B2B产品：周末流量自然下降 → 正常现象
2. B2C产品：周末应有提升 → 检查广告预算是否周末降低
3. 竞品周末促销 → 跟进促销策略

### 规则13：月初月末波动

**诊断标准**：月初或月末销量波动>30%

**可能原因**：
1. 消费者发薪周期 → 调整促销时间
2. 竞品月末冲量 → 避免正面竞争
3. 库存周期 → 确保库存充足

---

## 九、竞品对标问题

### 规则14：与竞品差距大

**诊断标准**：ACOS高于竞品均值20%以上

**对标维度**：
| 维度 | 诊断方法 | 优化方向 |
|------|----------|----------|
| 价格 | 对比竞品价格区间 | 调整定价策略 |
| 评论 | 对比评论数量和评分 | 积累评论 |
| Listing | 对比标题、图片、描述 | 优化Listing |
| 广告 | 对比广告类型和预算 | 优化广告结构 |

### 规则15：竞品价格战

**诊断标准**：竞品降价>10%且销量上升

**应对策略**：
1. 短期：设置优惠券或促销，不直接降价
2. 中期：优化Listing突出差异化价值
3. 长期：提升品牌溢价能力

---

## 十、预算问题

### 规则16：预算花不完

**诊断标准**：日预算使用率<70%

**可能原因**：
1. 关键词覆盖不足 → 拓展关键词
2. 竞价过低 → 提高竞价
3. 关键词不匹配 → 优化匹配方式
4. 时段限制 → 检查广告投放时段设置

### 规则17：预算过早花完

**诊断标准**：日预算在中午前花完

**优化建议**：
1. 增加日预算
2. 使用竞价调整，降低非高峰时段出价
3. 优化关键词，提高质量得分以降低CPC
4. 设置广告投放时段，集中在转化高峰

---

## 十一、异常值处理规则

### 规则18：ACOS异常高（>100%）

**处理方式**：
1. 检查是否有数据错误（如花费录入错误）
2. 检查是否有大额点击但无转化
3. 如确认为异常，标记并单独分析

### 规则19：CPC异常高（>$5）

**处理方式**：
1. 检查关键词竞争程度
2. 检查质量得分
3. 如确认为异常，降低竞价或暂停关键词

### 规则20：CVR异常高（>50%）

**处理方式**：
1. 检查是否有数据错误
2. 检查是否为品牌词搜索
3. 如确认为正常，分析成功因素并复制
