# 术语表

| 术语 | 定义 | 章节 |
|------|------|------|
| 天阶功法 | 从国家顶层设计出发寻找投资标的的研究方法 | ch04 |
| 地阶功法 | 回到利润、现金流、股本结构等基本功的价值投资方法 | ch05 |
| 玄阶功法 | 通过跨行业类比发现投资机会的逻辑 | ch06 |
| 邪修功法 | 短期博弈、情绪交易的投机逻辑 | ch07 |
| "求" | 商业模式的核心：谁有求于谁，决定利润分配权和估值层级 | ch02 |
| 三求 | 上游求+下游求+政府求，最优商业模式 | ch02 |
| 安全主线 | 所有投资逻辑嵌入的框架：能源安全/金融安全/药品安全/资源安全 | ch04 |
| 第一性原理 | 每个行业最关键的变量（铝=电，煤化工=煤价/转换比） | ch04 |
| 流动性估值陷阱 | 通过锁死流动性制造高估值假象 | ch02 |
| 布油组合(B+T) | 甲能源(煤→油)+布王(油→布)=煤→布，排除原油影响 | ch01 |
| 紫菜组合(Z+C) | Z矿业(有色)+C股份(珠宝)，对冲金价波动 | ch01 |
| 期权思维 | 以低价买入基本盘+免费获得远期增长期权 | ch04 |
| 模糊的正确 | 不追求精准数字，把握大方向的确定性 | ch08 |
| 一眼看胖瘦 | 快速识别企业好坏的能力，类似古玩圈"大开门" | ch08 |
| 弹性系数 | 单位利润对市值的拉动系数=边际增加利润÷市值×100 | ch04 |
| 扣非净利润 | 扣除非经常性损益后的净利润，更能反映主营业务 | ch05 |
| 经营性现金流 | 企业经营活动产生的现金净流量，验证利润含金量 | ch05 |
| PE(TTM) | 滚动市盈率，最近12个月每股收益计算的市盈率 | ch02 |
| PB | 市净率，股价与每股净资产的比值 | ch02 |
| PEG | 市盈增长比率=PE÷盈利增速 | ch02 |
| 股息率 | 每股分红÷股价×100%，估值的核心锚 | ch02 |
| 预期股息率 | 基于未来盈利预测计算的股息率 | ch02 |
| 除权 | 分红后股价下调，不是"把钱从左口袋放到右口袋" | ch05 |
| 股银模型 | 永久组合：50%银行股+25%生产资料+25%黄金 | SKILL |
| 股票锚 | 财富=手中持有的能产生正向现金流的优质生产资料数量 | ch12 |
| 现金锚 | 以账户法币净值衡量财富（应摒弃） | ch12 |
| 投资联想法 | 看好某行业不买终端，买入最上游资源或弹性最大的衍生概念 | ch05 |
| 只应对不预测 | 核心准则：不做市场预判，只根据已发生的事做应对 | SKILL |
| 确定性溢价 | 确定性本身就是稀缺品，稀缺的才有溢价 | SKILL |
| 平方差魔咒 | 连续投资期望收益为零的标的，收益会诡异地减少 | ch08 |
| 盲盒可视化 | 银行投资理论：不盯财报数据，盯盲盒里偶尔露出的可见部分 | ch08 |
| 经现归⺟⽐值 | 经营性现金流÷归母公司净利润，配合PB使用评估银行 | ch08 |
