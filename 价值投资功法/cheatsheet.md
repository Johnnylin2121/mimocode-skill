# 速查表

## 选股检查清单
- [ ] 商业模式"求"的层级？（三求/二求/一求/零求）
- [ ] 第一性原理是什么？（行业最关键的变量）
- [ ] 顶层设计指向哪里？（政策文件→行业痛点→标的）
- [ ] 预期股息率？（4%以上，最好5%+）
- [ ] 扣非净利润 vs 经营性现金流？（现金流应接近或超过扣非净利）
- [ ] 流通股本 vs 总股本？（差距大=流动性陷阱风险）
- [ ] 与同行相比估值如何？（PE分位值、PB分位值）

## 估值速查
| 方法 | 适用场景 | 要点 |
|------|---------|------|
| PE法 | 盈利稳定企业 | 纵比10年分位值+横比同行 |
| PB法 | 净资产与收益线性相关（如银行） | 门槛低，适合新手 |
| PEG法 | 年增速20-50%成长企业 | 辅助估值，不单独用 |
| 股息率法 | 所有高股息标的 | 最简单，跌得越多股息率越高 |
| DCF法 | 不推荐 | 贴现率微小差异导致估值天壤之别 |

## 仓位控制速查
| 规则 | 数值 |
|------|------|
| 持仓数量 | 5-6只（新手） |
| 单只上限 | 20%（银行除外） |
| 单只下限 | 5% |
| 加仓触发 | 跌幅超10% |
| 加仓节奏 | 每跌5%加1% |
| 止盈线 | 30%（新手） |
| 计划仓位 | 60%，空40% |

## 行业分析速查
| 行业 | 第一性原理 | "求"层级 | 关键变量 |
|------|-----------|---------|---------|
| 铝 | 电（铝=固体的电） | 电解铝二求 | 电价、铝价 |
| 煤化工 | 煤价/转换比/油价 | 二求 | 油煤差价 |
| 黄金流通 | 成交量 | 一求 | 金价、税收政策 |
| 有色（铜/金） | 矿 | 二求 | 金属价格 |
| 磷化工 | 磷矿 | 纯磷矿三求 | 矿价、品位 |
| 药品零售 | 店面数量 | 一求→二求 | 加盟店增速 |
| 银行 | 净资产 | 二求 | 净息差、股息率 |
| 地产 | — | 零求（最差） | 不推荐 |

## 投资主线速查
| 安全方向 | 路径 | 典型标的类型 |
|---------|------|------------|
| 能源安全 | 富煤少油→煤炭替代石油 | 煤制烯烃、电解铝 |
| 金融安全 | 黄金+银行 | 珠宝零售、银行股 |
| 药品安全 | 合规+规模效应 | 药品零售龙头 |
| 资源安全 | 磷矿稀缺+防止出口 | 磷矿企业 |
