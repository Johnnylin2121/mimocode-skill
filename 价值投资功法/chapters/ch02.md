# 第02章 股票的估值方法

## 核心观点
估值不是算术问题，而是层级问题。决定估值高低的不是公式，是"求"。

## "求"的商业模式层级
- **三求**（上游求+下游求+政府求）= 最优：茅台、苹果、英伟达
- **二求** = 次优：有色（上游求+政府求）、比亚迪（上游求+政府求）
- **一求** = 普通：95%以上企业
- **零求** = 最差：地产（上游是政府，下游不求，上游也不惯着你）

有色金属 vs 地产：都有周期，但有色比地产高两个层级。有色面粉价格基本稳定，面包有周期；地产面粉（土地）紧盯着面包价格，面包涨价面粉涨得更快。

## 五种估值方法
1. **PE法**：最常见，适用于盈利稳定的企业。纵比自己（10年分位值）+ 横比同行
2. **PB法**：适用于净资产与收益线性相关的行业（如银行），门槛低
3. **PEG法**：适用于年复合增速20%-50%的成长型企业，多用于辅助估值
4. **DCF法**：学术说法，实用性几乎为零（贴现率微小差异导致估值天壤之别）
5. **PD法**（市梦率）：适用于高科技企业，梦想越大估值越高

**总结**："PE为主，博取上限；PB为辅，保障下限；股息为先，进退自如；增速为次，稳定发展。"

## 流动性估值陷阱
通过锁死流动性制造高估值假象：
- 流通股本远小于总股本时估值扭曲
- 新股上市溢价高（流通盘占比太小）
- 银：60%被ETF锁死，实际可流通仅40%
- 二手房：也是通过限制流动性维持高估值

**警惕**：流动性陷阱 + 锚定心理 = "核动力收割机"。实际价值50元的东西炒到两三百元，散户盯着高价"抄底"，等来的是解禁公告和血亏。

## 估值案例（B公司）
- PE法：10年30分位值16.58倍，合理估值15倍PE以上
- 股息保障下限：预期分红0.8元，5%股息率对应股价下限16元
- 跌得越多，股息率越高，会有专吃股息的资金买入
