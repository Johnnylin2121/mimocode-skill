# 每日复盘工作流

## 核心原则

1. **预期-验证闭环**：每日复盘提出假设，次日早盘验证，形成闭环
2. **数据驱动**：所有判断必须有数据支撑，不用感觉代替事实
3. **市场状态优先**：复盘第一步是识别市场状态（趋势/震荡/轮动），再提出假设
4. **三日窗口**：复盘基于今天 + 前两个交易日，不多不少
5. **时间校验**：每个阶段开始前必须先获取当前日期/时间
6. **持仓评估**：复盘必须包含持仓标的评估，给出操作建议

---

## 市场状态识别

> **复盘第一步：先判断市场状态，再提出假设。**

| 状态 | 特征 | 量能 | 策略 |
|------|------|------|------|
| **趋势市** | 指数单边涨跌，板块持续 | >2.8万亿 | 顺势交易 |
| **震荡市** | 区间波动，有主线 | 2.5-2.8万亿 | 波段操作 |
| **轮动市** | 板块快速切换，一日游 | <2.5万亿 | **空仓观望** |

**判断方法**：
1. 指数趋势：近5日涨跌幅单边>3%=趋势
2. 量能：连续3日>2.8万亿=趋势市
3. 板块持续性：同一板块连续2日领涨=有主线

**轮动市陷阱**：
- ❌ 把超跌反弹当成趋势反转
- ❌ 频繁追涨杀跌
- ❌ 假设过于乐观

---

## 假设检验规则

### 假设必须包含

1. **验证标准**：具体的价格/涨跌幅/量能阈值
2. **对冲情景**：如果判断错了会怎样

### 假设格式

```markdown
### 假设X：[内容]
**依据**：[数据]
**验证标准**：[具体指标+阈值]
**对冲**：[如果错了会怎样]
```

### 禁止的假设

- ❌ "继续上涨/下跌"（无具体标准）
- ❌ "观察表现"（过于模糊）
- ❌ "超跌反弹可能演变为新一轮行情"（反弹≠反转）

---

## 工作流三阶段

### 阶段一：盘前准备（9:00前）

**步骤**：
1. **时间校验**：获取当前日期，确认前两个交易日
2. **读取复盘**：前两个交易日复盘文件的"明日关注点"和"可验证假设"
3. **读取财经早读**：提取关键信息
4. **审查持仓**：读取`交易体系/交易计划/当前持仓.md`
5. **拉取数据**：外围市场、大宗商品、A50期指
6. **制定盘前预测**：写入`交易体系/盘前预测/YYYY-MM-DD 盘前预测.md`

**盘前预测结构**：
```markdown
# YYYY-MM-DD 盘前预测
## 一、前日复盘回顾
## 二、今日核心事件
## 三、外围市场
## 四、观察清单
## 五、决策树
## 六、核心假设
## 七、操作建议
```

### 阶段二：盘中验证（10:00 / 13:00）

**步骤**：
1. 拉取实时数据（指数+个股+板块资金流向）
2. 逐条对照观察清单验证
3. 评估持仓标的
4. 更新盘前预测文件（添加"盘中验证 HH:MM"章节）

**盘中验证格式**：
```markdown
## 盘中验证 HH:MM
- 量能：推算收盘X.X万亿
- [方向]：[结论]（数据）
### 持仓评估
| 标的 | 价格 | 涨跌幅 | 评估 |
### 操作决策
```

### 阶段三：盘后复盘（15:00后）

**步骤**：
1. **时间校验**：确认今日日期
2. **拉取收盘数据**：指数+个股+板块资金流向+持仓标的
3. **读取前两个交易日复盘**
4. **更新前日复盘**：添加"今日验证"章节
5. **创建当日复盘**
6. **验证盘前预测**：对照收盘数据验证假设
7. **归档**：盘前预测移入存档，旧复盘移入存档

**复盘后操作**：
- 更新`交易体系/交易计划/当前持仓.md`的"今日评估"章节
- 归档盘前预测至`交易体系/盘前预测/存档/`
- 归档旧复盘至`交易体系/每日复盘/存档/`

## 数据获取

### 东方财富API

**指数行情**：
```
https://push2.eastmoney.com/api/qt/ulist.np/get?fltt=2&fields=f2,f3,f4,f6,f12,f14&secids=1.000001,0.399001,0.399006,1.000688,1.000300
```

**个股行情**：
```
https://push2.eastmoney.com/api/qt/ulist.np/get?fltt=2&fields=f2,f3,f4,f6,f12,f14&secids={市场代码}
```
市场代码：`1.600000`(沪市) / `0.000001`(深市)

**板块资金流向**：
```
https://push2.eastmoney.com/api/qt/clist/get?pn=1&pz=20&po=1&np=1&fltt=2&invt=2&fid=f62&fs=m:90+t:2&fields=f12,f14,f62,f184,f3
```

**字段映射**：
| 字段 | 含义 |
|------|------|
| f2 | 最新价 |
| f3 | 涨跌幅(%) |
| f6 | 成交额(元) |
| f12 | 股票代码 |
| f14 | 股票名称 |
| f62 | 主力净流入(元) |

## 常见错误

### 操作错误
1. **早盘验证写错位置**：应写入前日复盘的"今日验证"，不是当日复盘
2. **盘前预测未验证归档**：盘后必须验证并归档
3. **日期错误**：每个阶段开始前必须先获取当前日期

### 判断错误
4. **把超跌反弹当成趋势反转** → 反弹≠反转，下跌趋势中反弹是卖出机会
5. **在轮动市中频繁追涨杀跌** → 板块快速切换时，追着跑会被收割
6. **假设过于乐观，缺乏对冲** → 每个假设必须包含"如果错了会怎样"
7. **忽视量能信号** → 量价背离是强烈危险信号
8. **频繁切换关注方向** → 不要每天追"新主线"
9. **在防御型市场中找进攻机会** → 市场已经在防御，这是最大错配
