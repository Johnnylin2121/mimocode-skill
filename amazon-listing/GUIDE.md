# Amazon Listing Skill — 使用指南

## 简介

本 Skill 用于辅助 Amazon 卖家完成从竞品分析到 Listing 上架的完整工作流，涵盖：
- 竞品 Listing 爬取与核心关键词提取
- 标题 + 五点描述的撰写与关键词埋词
- 后台搜索关键词（Backend Search Terms）生成

支持任意 Amazon 站点（.com / .co.uk / .de / .co.jp / .com.mx 等），自动适配目标市场的语言风格。

---

## 触发方式

在与 AI 的对话中，提及以下任意关键词即可自动加载本 Skill：

> "亚马逊 listing" / "竞品分析" / "关键词" / "标题五点" / "后台搜索词"

**示例触发语**：
- "帮我分析这几个竞品链接的亚马逊 listing"
- "我要上架一个产品，先做关键词分析"
- "写一下这个产品的标题和五点"

## 五步工作流

### Step 0 — 确认目标站点

**AI 行为**：
- 默认以你提供的竞品链接所属站点为目标上架站
- 如果你希望上架到其他站点，需主动说明

**你需要做的**：
- 提供 1-5 个竞品链接（建议 3-5 个，分析更可靠）
- 竞品需与你目标上架的站点一致

---

### Step 0.5 — 获取产品图片（关键步骤）

**⚠️ 为什么图片是必须的？**

图片揭示产品的**实际规格**，决定哪些关键词适用：
- 端口类型和数量（USB 3.2、USB 3.0、USB-C、USB-A）
- 功率规格（45W、65W、适配器类型）
- 材质（铝合金、塑料）
- 设计特点（人体工学、紧凑型）
- 包含配件（线缆、适配器）

**没有图片的风险：**
- 提取的关键词可能对应产品没有的功能
- 错过产品的实际差异化优势
- 创建的 Listing 与真实产品不匹配

**你需要做的：**
- 提供产品图片（主图 + 详情图）
- 或提供产品文件夹路径
- 如果没有图片，提供详细的产品规格文字描述

**AI 会做什么：**
- 分析每张图片，提取产品规格
- 生成产品规格摘要表
- 用于后续关键词分析

---

### Step 1 — 核心关键词提取

**输入**：
- 产品规格（来自 Step 0.5 图片分析）
- 1-5 个竞品 Amazon 链接（同站点）

**AI 行为**：
1. 尝试用浏览器自动化爬取每个竞品的标题和五点
2. 若爬取失败，会请你手动复制粘贴竞品信息
3. 从竞品标题中提取**完整的搜索短语**（不是孤立单词）
4. 统计短语在竞品中出现的频率（流量指标）
5. 与产品规格匹配（匹配度）
6. 输出 Top 10-15 核心关键词

**⚠️ 关键审核点**：
- 核心关键词必须是**语义明确的搜索短语**
- ❌ 错误示例："usb"、"puertos"、"alimentación"（孤立单词，缺乏明确指向）
- ✅ 正确示例："hub usb 3.2"、"concentrador usb alimentado"（完整短语，语义明确）

**你需要做的**：
- 审阅关键词是否语义明确
- 确认关键词是否匹配产品实际规格
- 确认无误后说「确认」进入下一步

---

### Step 2 — 标题与五点撰写

**输入**：
- 确认后的核心关键词
- **产品主图**（AI 会视觉分析颜色/形状/材质/特征）
- **产品参数**（尺寸、重量、材质、颜色、配件清单、兼容性等）

**AI 行为**：
1. 按关键词权重分层埋词：
   - 前 3 个核心关键词 → 标题必须包含
   - 关键词 1-7 → 标题 + 五点均匀覆盖
   - 关键词 8-15 → 五点覆盖
2. 根据目标站点语言进行本地化润色
3. 遵守 Amazon 规则：标题 ≤200 字符、五点每点 ≤500 字符、禁用促销词
4. 每个五点聚焦一个主要卖点

**你需要做的**：
- 检查标题是否自然可读，避免关键词堆砌
- 确认差异化卖点是否被突出
- 确认无误后说「确认」进入下一步

---

### Step 3 — 后台搜索关键词

**输入**：
- 已确认的标题和五点文案

**AI 行为**：
1. 提取标题+五点中所有已使用的词，建立「排除集」
2. 从以下来源生成候选词：
   - 核心关键词中未完全嵌入 Listing 的部分
   - 同义词 / 常见别称
   - 常见错拼（如 bluetooth → blutooth）
   - 互补产品词（如手机壳 → 屏幕保护膜）
   - 目标市场其他语言词（如 US 站加西班牙语词）
   - 长尾词
3. 过滤：剔除排除集中的词、品牌名、促销词、ASIN
4. 组装：空格分隔，无逗号/分号，控制在 ≤249 bytes

**你需要做的**：
- 确认字节数未超限（249 bytes）
- 审阅是否有遗漏的重要流量词
- 如有需要，替换部分词后保存

## 重要规则与约束（Skill 内置）

| 项目 | 限制 |
|------|------|
| 标题长度 | ≤ 200 字符 |
| 单条 Bullet | ≤ 500 字符 |
| 后台搜索词 | ≤ 249 **bytes**（非字符） |
| 禁用词 | best, #1, cheap, guaranteed, amazing, free shipping 等 |
| 后台词分隔 | 仅空格，无逗号/分号 |
| 后台词重复 | 不得重复标题/五点中已出现的词 |

## 常见问题

### Q: 为什么不能只用孤立的单词作为关键词？
A: 孤立的单词如"usb"、"puertos"缺乏明确指向，会吸引不精准的流量。用户搜索时通常使用完整的短语，如"hub usb 3.2"或"concentrador usb alimentado"。

### Q: 如果没有产品图片怎么办？
A: 可以请用户提供详细的产品规格文字描述。但关键词分析的准确性会降低，建议尽量获取图片。

### Q: 竞品链接少于3个怎么办？
A: AI会提示分析可靠性降低，但仍会继续。建议尽量提供3-5个竞品链接。

### Q: 如何判断关键词是否"语义明确"？
A: 问自己：用户搜索这个词时，能明确知道他们想要什么产品吗？"usb"不能，但"hub usb 3.2"可以。

---

## 注意事项

1. **竞品链接数量**：建议 3-5 个。少于 3 个时 AI 会提示分析可靠性降低
2. **品牌排除**：AI 会自动排除品牌词，但如果有遗漏，请手动指出
3. **图片分析**：AI 会视觉分析主图。如果分析不准确，请以你的实际产品为准
4. **语言适配**：
   - .com (US)：美式英语，直接有力
   - .co.uk (UK)：英式拼写（colour, aluminium）
   - .de：德语，名词大写
   - .co.jp：日语敬体
   - .com.mx：墨西哥西班牙语
5. **关键词审核**：确认关键词是语义明确的搜索短语，而非孤立单词
